# Time-Series Behavior Lab

Exploration of temporal systems through simulation and analysis.

## Topics Covered

- Stable, noisy and transition systems
- Rolling mean and rolling standard deviation
- Correlation
- Autocorrelation and memory
- Random walks
- Mean reversion
- Oscillatory systems

## Tools

- Python
- NumPy
- Pandas
- Matplotlib
